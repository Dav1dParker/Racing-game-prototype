# Racing Game Prototype (Unity)

Прототип аркадной гоночной игры, созданный на Unity 6, с реалистичной физикой автомобиля, дрифта, ускорения, сцеплением с разными поверхностями, контрольными точками и таймером круга.

[Видео-превью](https://youtu.be/_Ur1qbtELIs)

[English readme](README.md)

---

## Возможности

### Физика автомобиля
- Реалистичная симуляция на основе Rigidbody и WheelColliders
- Настраиваемые типы привода: FWD, RWD, AWD
- Изменение угла поворота в зависимости от скорости
- Аэродинамическое сопротивление и трение
- Начало заноса по изменениею  бокового ускорения (G)
- Разное сцепление для разных поверхностей (асфальт, грязь, лед и т.д.)

### Система ускорения (Boost)
- Временное ускорение с регулируемой мощностью, длительностью и перезарядкой
- Возможность подзарядки через подобираемые бонусы
- Вибрация контроллера при ускорении
- Шкала прогресса перезарядки в UI

### Система кругов
- Подсчет прохождения кругов через контрольные точки, поддержка реверсивной вариации трассы
- Интерфейс с таймером текущего и лучшего круга
- Сохранение лучших результатов в JSON-файл

### Аудио
- Звуковые эффекты при ускорении, перезарядке и завершении круга

### Поверхности
- Класс SurfaceGripData задаёт коэффициенты трения для разных тегов поверхности
- Определение типа поверхности через коллайдеры колес

### Визуальные эффекты
- Камера Cinemachine для плавного следования и динамического изменения FOV в зависимости от скорости
- Стоп-сигналы при торможении
- Подбираемые бонусы с анимацией (ускорение, заряд) с таймером возрождения

```

    ├── Scripts/
    │   ├── Audio/               # Звук

    ├── Prefabs/
    ├── Scenes/
    └── Sounds/
```

## Технологии

| Компонент | Описание |
|------------|-----------|
| Движок | Unity 6 (6000.2.9f1) |
| Ввод | Unity New Input System |
| Камера | Cinemachine |
| Физика | WheelColliders + Rigidbody |
| Сохранения | JSON через System.IO |

---


## Сохранения

- Лучшие времена кругов сохраняются в JSON-файле:
  
  ```/RacingGamePrototype/Assets/BestLapTimes.json```
- Создаётся автоматически при первом запуске
- Сохраняется между сессиями и обновляется при установлении нового рекорда

## Управление

| Действие | Клавиатура | Геймпад |
|----------|-------------|----------|
| Газ | W | RT |
| Тормоз / Задний ход | S | LT |
| Поворот | A / D | Левый стик |
| Ускорение | Shift | A |
| Респавн | R | Start |
| Реверс трассы | O | Y |

## Как запустить

1. **Клонируйте репозиторий**

   ```git clone https://github.com/Dav1dParker/Racing-game-prototype```

2. Откройте проект в **Unity 6000.2.9f1** или новее.
3. Откройте сцену:

   ```Assets/_CardGamePrototype/Scenes/Track.unity```
4. Нажмите **Play**.

### Или

1. [Скачать архив](https://disk.360.yandex.ru/d/wLqe-QuUWAdMvA)
2. Распакуйте архив
3. Запустите ```RacingGamePrototype.exe```
